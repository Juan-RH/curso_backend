# Curso Backend - APIs REST

Repositorio del curso de desarrollo de aplicaciones web backend con Node.js y Express.js.

## Objetivos del curso

- Desarrollar aplicaciones web del tipo API REST
- Dominar patrones de arquitectura cliente-servidor
- Utilizar Node.js y Express.js para aplicaciones backend
- Gestionar paquetes y módulos nativos y de terceros
- Implementar servicios dinámicos en la nube (Firestore)
- Desarrollar capas de autenticación
- Integrar conocimientos en un proyecto final
